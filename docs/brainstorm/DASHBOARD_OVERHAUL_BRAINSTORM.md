## Specific Widgets to Build

### 1. Company Health Cards (Grid)

Small cards showing per-company ticket health:

- Company logo (we just built this!)
- Open count with color coding
- Urgent badge (red pulse if > 0)
- Overdue badge (yellow if > 0)
- Mini sparkline (7-day trend)
- Click â†’ drills into company tickets

### 2. Live Activity Feed

Real-time ticker of recent activity:

- New tickets created
- Tickets resolved
- Status changes
- Priority escalations
- Color-coded by priority
- Click â†’ opens ticket modal

### 3. Attention Needed Panel

Actionable alerts only:

- Overdue tickets (sorted by age)
- Unassigned high-priority
- SLA breaches
- Stalled tickets (no update in X days)
- Each item clickable

### 4. Agent Workload Bar

Horizontal bars showing:

- Agent name + avatar
- Assigned ticket count
- In-progress vs waiting
- Click â†’ shows agent's tickets

### 5. Created vs Resolved Chart

Simple line/bar chart:

- Last 7/14/30 days
- Created line vs Resolved line
- Gap = growing backlog (bad)
- Filters by company

### 6. Priority Distribution Donut

Quick visual of ticket mix:

- Critical/Urgent/High/Normal/Low
- By company or overall
- Click segment â†’ filters tickets

### 7. Response Time Gauge

Circular gauge showing:

- Current avg first response time
- Target (SLA) line
- Good/Warning/Bad zones
- Per company comparison
